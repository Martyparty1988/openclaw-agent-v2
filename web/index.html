Every message goes through a pipeline: normalized by the Router, interpreted by the MetaAgent, planned by the Planner, executed by the Executor — all with persistent session Memory.

⚙️

💾

🔍
Analyze

✏️
Refactor
Generates improved versions of high-priority files. Keeps .bak backups before overwriting.
🧪

Runs npm test and syntax checks. Only applies changes if tests pass.
📦

git add → commit "🤖 self-improve: ..." → push origin main. Fully autonomous.

// whatsapp commands

Talk to it
like a developer.

> analyze <what>
Analyze a file or codebase. No changes made — pure analysis with actionable findings.
> plan <task>
Generate a structured step-by-step execution plan for any task.
> execute <task>
Run the agent async. You get an immediate ack, then a notification when it's done.
> improve
Trigger the full self-improve cycle: analyze → refactor → test → git push.
> chat <message>
Free-form conversation with full session memory across messages.
> reset
Clear your session memory. Start a fresh conversation context.
